# Jazarift Capital Website (Next.js)

A modern, responsive VC firm website redesign built with Next.js App Router and TypeScript.

## Features

- Sticky glassmorphism navigation with quick anchors
- High-conversion hero section with metrics and dual CTAs
- Investment thesis, portfolio highlights, and value-add sections
- Contact CTA band and lightweight footer
- Fully responsive layout for mobile and desktop

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- CSS Modules

## Local Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build & Production Check

```bash
npm run lint
npm run build
npm run start
```

## Main Files

- `app/page.tsx` – website sections and content
- `app/page.module.css` – full visual system and responsive styles
- `app/layout.tsx` – metadata and app shell
